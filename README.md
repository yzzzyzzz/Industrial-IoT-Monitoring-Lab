# Industrial-IoT-Monitoring-Lab
Wokwi ESP32 水溫警告模擬系統
# 🏭 工業級監控與維運自動化實戰 (IoT & System Monitor)

這個專案記錄了我將 **廠務水處理工程師** 的實務經驗，轉化為 **Linux 維運與 IoT 自動化** 技術的開發過程。

## 🚀 核心專案內容

### 1. ESP32 水溫告警模擬系統 (`/iot_simulation`)
* **功能**：模擬水處理設備感測器，當水溫超過 35°C 時自動觸發告警。
* **技術**：MicroPython, Wokwi 模擬平台, MQTT 協定。
* **實戰成果**：[點此查看 Wokwi 線上模擬運行](https://wokwi.com/projects/322577683855704658)

## 🛠️ 技術成長與問題解決 (Log & Troubleshooting)

在開發過程中，我遇到了以下關鍵挑戰，並展現了維運工程師的排錯能力：

1.  **連線不穩定問題 (ECONNRESET)**：
    * **現象**：MQTT 傳輸時發生 `OSError: [Errno 104]`。
    * **解決**：分析 Log 後，實作了「斷線重連機制 (Auto-Reconnect)」，確保監控系統在網路波動時能自動恢復，符合工業穩定性要求。

2.  **語法縮排錯誤 (IndentationError)**：
    * **現象**：在加入複雜邏輯時發生 Python 縮排對齊問題。
    * **解決**：重新梳理程式碼區塊邏輯，提升代碼可讀性與結構嚴謹度。
